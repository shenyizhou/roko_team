# 属性评分系统

## 核心原则
- 游戏meta: 每只精灵带1个防御技+状态技，输出技能少，学习面受限
- 弱点主要怕愿力但防不住 → 弱点扣分小
- 抵抗价值 > 弱点扣分（抵抗是联防核心）

## 评分公式
```
defense_raw = sum(每个攻击属性:
  if 抵抗(<=0.5): +7 * 属性权重
  if 弱点(>=2.0): -5 * 属性权重
)
type_defense = max(defense_raw, 0)  # 差属性=0，不扣分
attr_score = type_defense + max(type_synergy, 0)
```

## 属性权重
- T1 (热门): 一般, 翼, 水, 地, 机械, 火 — w=2.0
- T2 (次热): 光, 恶魔, 冰, 电, 武 — w=1.5
- 其他: w=1.0

## 目标分数范围
- 顶级联防（如寂灭骨龙 龙+幽）: ~100
- 差属性（如单虫）: ~0-5
- 中等属性: ~30-60

## 注意
- 游戏无0x免疫，最低1/3抵抗
- JSON数据用"0"/"1"/"2"简化键，无法区分1/2vs1/3或2xvs3x
