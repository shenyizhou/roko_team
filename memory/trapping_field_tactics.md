# 捕获战术 & 场地战术

---

## 捕获战术（Trapping）

### 核心机制
**抓人**：强制对方无法换人，然后用克制的精灵秒杀。

### 实现方式
1. **地系流沙**：直接禁止换人
2. **压力捕获**：通过压力和先读，让对手换人时刚好换上克制的精灵
   - **例**：龙息帕尔对位寂灭骨龙 → 寂灭骨龙压力很大想换绒光优优 → 预判换人上帕帕斯卡用机械系技能秒杀
   - **例**：彩虹独角兽给压力 → 雪影冰灵登场 → 帕帕斯卡捕获

### 关键精灵
- **圣羽翼王**：迅捷闪击/水刃 → 场下压力极大
- **帕帕斯卡**：迅捷大威力技能 → 场下压力极大
- 这两只精灵的场下压力是捕获战术的核心动力

### 算法应用
1. 拥有迅捷技能的精灵需要大幅加分（场下压力）
2. 帕帕斯卡和圣羽翼王的组合需要额外协同加分
3. 地系精灵（流沙技能）可以考虑捕获战术加分

## 场地战术（Field）

### 核心机制
通过**印记和天气**改变check和counter的对位关系。

### 基本结构
- **至少1只场地释放者**（起点）
- **2只以上收益的终点**
- **需要注意对方工兵**

### 工兵精灵（破坏场地）
- **尖嘴狐仙**：焚烧烙印（驱散双方印记）
- **利灯鱼**：倾泻（驱散双方印记）
- 如果工兵能counter起点 → 战术很难实施

### 应对工兵的策略
1. **带终点counter工兵**：至少1只能counter工兵的终点
2. **风墙精灵**：通过换人迅捷防御阻挡倾泻并施加击杀工兵的压力
   - **迷迷箱怪**：啮合传递
   - **翠顶夫人**：力量增效

### 核心规律
- 如果工兵能counter终点 → **起点必须counter工兵**
- 如果工兵能counter起点 → 需要终点counter工兵

### 算法应用
1. 检测队伍的工兵对策能力
2. 场地释放者 + 收益者数量检测
3. 工兵防御能力检测
