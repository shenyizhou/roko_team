# 先读（预判）概念

## 定义
先读 = 根据对方可能会做什么操作，提前预判来做更加应对的操作

## 压力来源
### 场上压力
- **清唱手/有强化的高速清场手**
- 一旦放任对方强化就会导致队伍整个被推的场景
- 必须立刻处理，否则崩盘

### 场下压力
- **迅捷手**带来的压力
- 代表精灵：
  - **圣羽翼王**：迅捷闪击、迅捷水刃
  - **帕帕斯卡**：迅捷大威力技能

## 迅捷机制详解
- 在换上来的那一回合就触发第一个有迅捷效果的技能
- 如果速度比对方快，就能实现在对手使用技能前先手释放
- **实战应用**：
  - 配合**闪击**：直接秒杀对手
  - 配合**水刃**：应对状态技能 → 水刃变成0费 → 后续圣羽翼王可以灵活轮转顺带打水刃伤害

## 评分应用
1. 有迅捷技能的精灵需要大幅加分（尤其是高速的）
2. 圣羽翼王的核心价值在于迅捷技能的轮转压制
3. 迅捷技能本身的评分需要提高
4. 组队时需要考虑迅捷压制力
