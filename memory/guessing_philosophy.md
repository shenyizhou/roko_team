# 猜拳容错与复合战术

## 核心哲学
**洛克王国世界里没有那么多适合先读的场景，大部分都是胜率五五开的猜拳场景。**
重要的不是赢下某一次猜拳，而是：
1. 提供足够的**容错空间**减少猜拳失败的损失
2. 尽可能地**多创造猜拳的机会**

## 非冗余战术的两个关键要素

### 1. 起点精灵的counter集中度
- 如果起点精灵同时被对方**多只精灵counter** → 我方无法预判对方会上哪只 → 失去捕获机会
- **原则**：起点精灵的counter面越窄，先读成功率越高

### 2. 起点精灵的压力值
- 如果起点精灵给对方的压力不够大 → 对方可能不会换人
- **原则**：起点精灵必须有足够的反制能力，迫使对方必须换人

---

## 冗余战术与非冗余战术的复合

### 思路一：非冗余创造 → 冗余收益
通过控制/捕获/炮台等非冗余战术创造安全登场机会，再用定向施压打开局面。

### 思路二：冗余创造 → 非冗余扩大
通过定向施压逼出counter位，再用捕获/炮台等非冗余战术扩大优势。

---

## 算法评分哲学（关键转变）

### 从"最强队伍"到"最多机会"
- ❌ 旧思路：计算单次猜拳胜率，找最高分的队伍
- ✅ 新思路：计算队伍能创造多少安全猜拳机会，找容错率最高的队伍

### 容错率评分维度
1. **起点精灵counter面窄**：被少数精灵counter → 预判准确率高 → 加分
2. **起点精灵压力大**：对counter位有足够威胁 → 对方被迫换人 → 加分
3. **猜拳机会多**：队伍中多对精灵能形成猜拳关系 → 加分
4. **猜拳失败损失小**：联防能力兜底 → 减罚分
