# 炮台战术（非冗余战术）

## 定义
**炮台**：高速清场手或强化后对对手威胁极大的精灵，如龙息帕尔。
**炮台战术核心**：为炮台创造安全登场条件。

## 战术流程
1. 利用check/counter驱使对手换人
2. **先读**对方换人的时机
3. 换上克制对手的炮台
4. 炮台安全登场 + 对手换人 = **多一次免费强化**

**本质**：通过场上压力和场下压力**迫使对手换人**，从而给炮台创建登场窗口。

## 分类

### 单炮台
- 1只炮台 + 5只支援
- 支援精灵为炮台创造安全登场条件

### 多炮台
- 进攻轴里有2只（少见3只）炮台精灵
- 机制：炮台A给对手造成压力 → 逼对手换人 → 炮台B安全登场
- 两只炮台互相创造登场窗口

---

## 算法应用
1. **炮台识别**：高速（spd>120）+ 高输出 或 有强化技能 + 高输出
2. **多炮台加分**：检测队伍中有2+炮台且能互为冗余
3. **压力评估**：场上压力（强化清场手）+ 场下压力（迅捷手）
4. **炮台安全度**：炮台有多少check/counter的掩护
